# 🎟️ FlexBot Ticket
FlexBot Ticket is an open source ticket bot for [Fluxer](https://fluxer.app/) written in Java.
To add the bot to your server, click on this link: https://web.fluxer.app/oauth2/authorize?client_id=1524925097205309440&scope=bot&permissions=268435472

## ⚠️ Currently in development
The bot is currently in development.
It may contain bugs and may lack features.
For any bug reports or feature requests, please open an issue on GitHub or contact me on my [Fluxer Community](https://fluxer.gg/diZjBcsq).
This bot is made with [JFA](https://github.com/Fluxer-JFA/JFA) and [J4Fluxer](https://github.com/why-shiro/J4Fluxer).

## 📃 Commands
Here are all the bot commands:
- `/help`: Shows all available commands
- `/ticket open [title]`: Open a ticket with an optional title.

## 💾 Installation

To add the bot to your server, you can use the [invite link](https://web.fluxer.app/oauth2/authorize?client_id=1524925097205309440&scope=bot&permissions=268435472).
If you want to self-host the bot, you can ask for help in my [Fluxer Community](https://fluxer.gg/diZjBcsq).

### Needed permissions
Here are all the permissions the bot needs in case you want to self-host FlexBot Ticket:
- Manage Channels
- Manage Roles
